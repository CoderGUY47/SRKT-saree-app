## ✨ Overview

**SRKT** is a premium B2B e-commerce platform custom-built for wholesale saree operations, catalog showcase, and client-vendor interaction. It serves as a high-fidelity demonstration showing how traditional clothing wholesalers can digitize their distribution channels.

Wholesale clients can browse rich categories of sarees, filter by fabric or collections, curate wishlists, build purchase carts, and submit structured bulk inquiries directly to vendors. Administrators are equipped with a back-office panel to monitor enquiries, manage catalogs, update parameters, and inspect buyer metrics in real time. Built using **Flutter** and **Material Design 3**, the app features a warm, luxury-centric design language matching premium Indian apparel brands.

## ❌ The Problem & ✅ The Solution

> **Traditional wholesale textile operations suffer from communication bottlenecks.**
> Wholesalers rely on PDF catalogs, manual messaging, and unstructured lists, leading to order inaccuracies and delays.

| ❌ The Problem | ✅ SRKT's Solution |
| :--- | :--- |
| **Outdated Catalog Sharing**: Sending heavy PDF brochures that go out of date instantly. | **Live Digital Showcase**: Synchronized visual grid catalog showing active collections. |
| **Messy Bulk Inquiries**: Clients submitting lists over text without sizing, quantity, or ID. | **Interactive Cart Checklist**: Direct checkout/enquiry building with precise MOQ specifications. |
| **High Friction Communication**: Extra steps to initiate transactions or request pricing details. | **Built-in WhatsApp API Routing**: Pre-formats your inquiry checklist and opens WhatsApp in one click. |
| **No Central Admin Registry**: Hard to track active buyers and past incoming wholesale inquiries. | **Back-Office Panel**: Dedicated tab to log, review, and follow up on client enquiries. |

## 🚀 Key Features

* **🎨 Luxury UI/UX Theme**: Outfit/Inter typography, warm background shades, clean borders, and smooth transitions matching luxury apparel brands.
* **📦 Product Catalog & Filters**: Saree collections categorized by fabric (Silk, Banarasi, Organza, Georgette) with quick searching.

* **📊 Back-Office Dashboard**: Visual admin counters for Total Enquiries, Registered Users, and active parameters.

## 🛠️ Developer Setup

### 1. Verification
Ensure your local SDK paths (Java 17, Flutter) are set up. Verify with:
```cmd
flutter doctor
```
*(No Android Studio installation is required).*

### 2. Running Locally on Emulator
1. Start your local emulator:
   ```cmd
   %LOCALAPPDATA%\Android\Sdk\emulator\emulator.exe -avd medium_phone -no-snapshot-load
   ```
2. Navigate and run the app:
   ```cmd
   cd SRKT
   flutter run -d emulator-5554 --no-pub
   ```

### 3. Compilation
To build target-specific release APKs:
```cmd
cd SRKT
flutter build apk --release --split-per-abi
```
